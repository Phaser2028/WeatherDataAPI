# WeatherDataAPI
RestAPI приложение для добавления датчика прогноза погоды и его измерений.

## При отправке GET-запроса по адресу `/sensors` отобразится список из всех датчиков
![bandicam 2023-07-29 17-17-57-854](https://github.com/Phaser2028/WeatherDataAPI/assets/43641188/066c51d7-273e-4cc7-bedf-8fc74057caf8)

## При отправке POST-запроса по адресу `/sensors/registration` данными о датчике в формате JSON, добавится новый датчик
![bandicam 2023-07-29 17-24-25-103](https://github.com/Phaser2028/WeatherDataAPI/assets/43641188/176440e7-29d7-4fe6-abb3-e1f81d5ddb58)
